# chat™ app

a vibe coded swiftui ios/ipados/macos app for https://chattm.app

check out https://github.com/emmameowss/chattm, the site this app is based on that's not as vibe coded as this
### install guide (ios/ipados)
i cannot provide IPAs for this because that requires a developer license that I don't have, so you'll have to compile it yourself, built using xcode 27 beta so you'll probably need that as well as an apple silicon mac

clone the repository, open it in the xcode beta, connect your phone, change the development signing certificate to your own, press run (dev license limits apply of course)

i'm not aware of a way you can install this into livecontainer or something similar

### install guide (macos)
download the DMG, put the app into the applications folder, allow it in privacy & security in settings
